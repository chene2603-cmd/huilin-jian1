# 《玄隐义客纪》架构蓝图

## 终极愿景
北宋汴京开放世界 · 动态NPC AI · 分支任务 · 国风渲染 · 内存安全 · 热更新

## 目录规划（按蓝图生长）
- core/          → 蓝图：底层内核（内存池、数学库、汇编、工具）
- engine/render/ → 蓝图：3D渲染（图形、模型、相机、光照、国风材质）
- engine/world/  → 蓝图：世界数据（汴京章节、NPC、任务、物品）
- engine/logic/  → 蓝图：游戏逻辑（场景、NPC、任务、战斗、经济）
- tools/         → 蓝图：运维脚本（编译、打包、CI、补丁管理器）

## 入口
main.c → engine_init() → [未来窗口初始化] → 主循环 → engine_shutdown()
